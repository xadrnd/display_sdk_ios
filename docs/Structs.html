Toast Style
  • ToastStyle

    ToastStyle instances define the look and feel for toast views created via the makeToast methods as well for toast views created directly with toastViewForMessage(message:title:image:style:).

    @warning ToastStyle offers relatively simple styling options for the default toast view. If you require a toast view with more complex UI, it probably makes more sense to create your own custom UIView subclass and present it with the showToast methods.
